НАВЫКИ: Переменные и типы данных

ВОПРОС:
Переменная в Python — это имя, которое указывает на область памяти, где хранится объект. В Python нет необходимости указывать тип переменной заранее, язык автоматически определяет тип во время выполнения на основе присвоенного значения.

Пример:
```python
x = 42      # целое число
y = "text"  # строка
```
Переменные создаются с помощью оператора присваивания `=`, а не с помощью ключевых слов.

Утверждение:
**В Python переменные создаются с помощью ключевого слова `var`.**

ВАРИАНТЫ:
A) Верно
B) Неверно

ПРАВИЛЬНЫЙ: Неверно

```python

```

Утверждение:
**В Python переменные всегда сохраняют тот тип, с которым были созданы.**

ВАРИАНТЫ:
A) Верно
B) Неверно

ПРАВИЛЬНЫЙ: B
